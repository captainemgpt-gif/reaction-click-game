# GAME_PLAN.md

## 1. 게임 제목
강아지 간식 잡기

## 2. 게임 한 줄 소개
화면에 나타나는 간식을 강아지가 놓치기 전에 빠르게 클릭해 점수를 올리는 귀여운 반응속도 게임.

## 3. 핵심 플레이 방식
- 게임 시작 버튼을 누르면 간식이 화면의 랜덤 위치에 나타남
- 플레이어는 제한 시간 안에 간식을 클릭함
- 클릭 성공 시 점수 +1, 간식은 새 위치에 다시 나타남

## 4. 화면 구성
- 상단: 현재 점수, 남은 시간
- 중앙: 플레이 영역(강아지, 간식 표시)
- 하단: 시작/재시작 버튼, 간단한 안내 문구

## 5. 점수 규칙
- 간식 1회 클릭 성공: +1점
- 클릭하지 못하고 시간이 지나 간식이 사라지면 점수 변화 없음

## 6. 타이머/종료 조건
- 전체 게임 시간: 38초
- 시간이 0초가 되면 게임 종료
- 종료 시 최종 점수와 재시작 버튼 표시

## 7. 그래픽 표현 방식
- 외부 이미지 사용 여부: 사용하지 않음(초기 버전)
- 강아지/간식/배경 표현 방식:
  - 강아지: 이모지(🐶) 또는 간단한 원형 캐릭터
  - 간식: 이모지(🦴) 또는 작은 색상 버튼
  - 배경: 단색 또는 부드러운 파스텔 톤

## 8. 초보자에게 쉬운 이유
- 규칙이 단순하고 조작이 클릭 1가지뿐임
- HTML/CSS/JavaScript 기본 기능만으로 구현 가능
- 이미지 파일 없이도 바로 실행 가능

## 9. 파일 구성 제안
- `index.html`: 게임 화면 구조
- `style.css`: 레이아웃/색상/애니메이션(간단)
- `script.js`: 점수, 타이머, 랜덤 위치 로직
- `GAME_PLAN.md`: 기획 문서(기준 문서)

## 10. 구현 범위(이번 버전에서 할 것 / 나중에 할 것)
- 이번 버전에서 할 것:
  - 단일 화면 게임
  - 점수/타이머/재시작 기능
  - 강아지/간식 기본 시각 표현
- 나중에 할 것:
  - 난이도 증가(속도 변화)
  - 최고 점수 저장
  - 사운드 효과 및 간단한 애니메이션 강화
